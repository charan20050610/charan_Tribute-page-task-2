Tribute Page – Dr. A.P.J Abdul Kalam

This project is a Tribute Page created using HTML and CSS to honor Dr. A.P.J Abdul Kalam, known as the Missile Man of India. The webpage highlights his inspiring life, achievements, and contribution to India’s space and missile programs.

The page features:

A clean and responsive design
Profile image section with title and subtitle
About section describing his life and achievements
Achievement list for quick information
Inspirational quote section with attractive styling
Simple UI using HTML and CSS only
